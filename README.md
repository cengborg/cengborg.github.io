# Chris Engborg Personal Site

This is a plain static GitHub Pages site with separate pages for the home page, research, and CV.

## Files

- `index.html` is the compact home page.
- `research.html` contains research cards, video embed slots, and the text animation canvas.
- `cv.html` embeds `assets/Engborg_CV_5-1-26.pdf`.
- `styles.css` controls the white, minimal layout.
- `script.js` renders optional video cards and the animated text canvas on pages that include them.


For YouTube embeds in `script.js`, use:

```text
https://www.youtube.com/embed/VIDEO_ID
```

For Vimeo embeds, use:

```text
https://player.vimeo.com/video/VIDEO_ID
```

## Publish

Push the repository to GitHub, open repository settings, enable GitHub Pages from the main branch
and root folder, then visit the published URL after deployment finishes.
